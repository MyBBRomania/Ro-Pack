Introduction

In order to satisfyas many users as possible, MyBB is available in a multitude of languages. By default MyBB is installed with the American English language files, but thanks to many volunteers, MyBB is also available in other languages. This language pack will allow you to use MyBB in Romanian.


Installation

The Romanian text is stored in language files, which are packed in an archive file together with this Readme and the Romanian translation of this Readme. These language files are divided amongst several folders. As of version 1.8.0, there are no images distributed in this package, as every button is created in HTML.

This Readme and the Romanian one are in the main folder. In the Upload subfolder, you can find a folder called romanian a file called romanian.php, which offers the basic informations about the transaltion. The files containing the translations of the text for the front-end are in a subfolder called romanian and the files containing the translations of the text for the back-end are in a subfolder called admin.

You will have to create the same folder structure on your server and place the language files in those folders. Below are extensive instructions on how to do this. We will assume you have already extracted the contents of the archive to your computer, which means the files are already available and the folder structure is as it should be. We will also assume you know how to use an FTP program to connect to your server. The tasks below can be executable with any FTP program.

  1. Open your FTP program and connect to your server.
  2. Navigate with your FTP program to the folder on your server in which you installed MyBB.
  3. Navigate to the folder on your server called inc and to the folder called languages, which will be located in the folder called inc. In the folder languages you should see a folder named english and a file named english.php.
  4. Copy the file romanian.php from your computer to this folder on the server. It should then be present in the languages, folder.
  5. After that, copy the folder called romanian from your computer to the folder languages on the server.

You have now uploaded all necessary files to the server. Read the following paragraphs for more information on how to enable the Romanian language on your board.


Usage

After uploading the required files you can use the Romanian translation for your copy of MyBB. We will hereafter discuss how to use the Romanian translation for the front-end of the board, how to use the Romanian translation for back-end of the board and how to set the Romanian translation as the default language for your board. The default language will be used by guests and users who haven't set a language preference.

  • To use the Romanian language for the front-end of the board, a user should go to the User CP and click Edit Options. Then he can select Romanian as "Board Language" below "Other options".
  • To set the Romanian language as the default language for your board, an administrator should go to the Admin CP and click the Configuration tab. He will see "General Configuration" at the top of the list. He should click on that title. Then he can select Romanian below "Default Language" and click "Save Settings" to submit the changes.
  • To use the Romanian language for back-end of the board, an administrator should go to the Admin CP and click the Configuration tab. At the bottom of the list he will see "Control Panel Preferences (Global)". He should click on that title. Then he can select Romanian below "Control Panel Language" and click "Save Settings" to submit the changes.

Additional information

If you wish to contact the authors of this translation, please go to the Admin CP and click the Configuration tab. After that, click Languages to view the Languages module. There you will find a list of installed language packs and a link to the authors' websites. You can click the links to contact the authors of this language pack. Or you can go to the MyBB Romania forum, where you will meet the staff and the users of the community, make new friends and get help for any problem you encounter with your forum.
