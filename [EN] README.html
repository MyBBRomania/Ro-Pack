Introduction

By default MyBB is installed with the American English language files, but thanks to the many volunteers, MyBB is also available in other languages. This language pack will allow you to use MyBB in Romanian.


Installation

The Romanian text is stored in language files, which are packed in an archive file together with this Readme and the Romanian translation of this Readme. These language files are divided among several folders. As of version 1.8.0, there are no images distributed in this package, as every button is created in HTML.

This Readme and the Romanian one are in the main folder. In the Upload subfolder, you can find a folder called romanian a file called romanian.php, which offers the basic informations about the transaltion. The files containing the translations of the text for the front-end are in a subfolder called romanian and the files containing the translations of the text for the back-end are in a subfolder called admin.

You will have to create the same folder structure on your server and place the language files in those folders. Below are extensive instructions on how to do this. We will assume you have already extracted the contents of the archive to your computer, which means the files are already available and the folder structure is as it should be. We will also assume you know how to use an FTP client to connect to your server. The tasks below can be executable with any FTP client.

  1. Open your FTP client and connect to your server.
  2. Navigate with your FTP client to the folder on your server in which you installed MyBB.
  3. Navigate to the folder inc and then to the folder languages (found in inc). In the folder languages you should see a folder named english and a file named english.php.
  4. Copy the file romanian.php and the folder romanian from your computer to the languages folder on the server.

You have now uploaded all the necessary files to the server. Read the following paragraphs for more information on how to enable the Romanian language on your board.


Usage

After uploading the required files you can use the Romanian translation for your copy of MyBB. We will now see how to use the Romanian translation for the front-end of the board, how to use the Romanian translation for back-end of the board and how to set the Romanian translation as the default language for your board. The default language will be used by guests and users who haven't set a language preference.

  • To use the Romanian language for the front-end of the board, the user should go to the User CP and click Edit Options. Then they can select Romanian as the Board Language below Other options.
  • To set the Romanian language as the default language for your board, the admin has to access the Admin CP, enter the Configuration tab and access the General Configuration setting group. The Default Language option will be found there.
  • To use the Romanian language for back-end of the board, has to access the Admin CP, enter the Configuration tab and and access the Control Panel Preferences (Global) setting group. The Control Panel Language will be found there.
